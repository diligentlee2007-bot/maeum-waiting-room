## 한 줄 소개

마음이 힘든 사람이 **1분 안에 '도움을 청하는 한 문장'을 만들고** 공식 상담 기관으로
연결되도록 돕는 공익 웹사이트.

## 왜 만들었나 (문제의식)

기존의 자살예방·정신건강 안내는 대부분 '전화번호 목록'에 그친다. 하지만 정작 힘든
사람은 *"이 정도로 도움을 청해도 되나"* 하는 망설임, *"전화로 말을 어떻게 꺼내지"* 하는
막막함 앞에서 멈춘다. 정보가 없어서가 아니라 **도움을 청하는 첫 문장과 첫 행동의 문턱이
높아서** 연결되지 못한다고 보고, 그 문턱을 낮추는 것을 목표로 했다.

## 차별점 / 장점

- **번호 모음이 아니라 '문턱 낮추기'에 집중** — 지금 마음 상태를 고르면, 상태에 맞는
  위로 문구·도움 요청 문장·연결 수단이 달라진다.
- **전화 외 대안 제공** — 말로 설명하기 어렵거나 곁에 가족이 있는 사람을 위해 채팅·문자(SNS)
  상담 카드를 함께 안내.
- **윤리적 안전 설계(Safety by design)** — AI 상담·자살 위험도 점수화·로그인·데이터 저장·
  게시판을 *의도적으로 배제*. 자가검진도 자체 게재하지 않고 공식 페이지로만 연결.
- **공식 기관만 연결** — 109·1577-0199·1388·1588-9191·119, 마들랜(자살예방 SNS 상담),
  국가정신건강정보포털 자가검진·센터찾기 등. 모든 외부 링크 실제 작동 검증.

## 사용자 흐름 (2단계)

1. **지금 마음 상태 고르기** (4개)
2. 고른 마음에 맞춰 **위로 문구 · 도움 요청 문장 · 연결 카드가 달라지는 맞춤 결과**

| 상태                              | 맞춤 결과의 중심                                                      |
| --------------------------------- | -------------------------------------------------------------------- |
| 그냥 너무 힘들어요                | 위로 + "이 정도로 도움받아도 됩니다" + 부담 낮은 문장 + 채팅 상담     |
| 누군가에게 말하고 싶어요          | 친구·가족·멘토별 도움 요청 문장 생성 + 복사 + 상대 응답 안내          |
| 병원이나 상담을 알아보고 싶어요   | 공식 상담전화·채팅·자가검진·센터찾기 기관 카드 모음                   |
| 지금 혼자 있으면 위험할 것 같아요 | 119·109 즉시 연결 우선 + 곁에 있어줄 사람에게 보낼 문장 + 채팅 상담   |

## 기술 스택

Next.js 16 (App Router) · TypeScript · Tailwind CSS v4 · Vercel 배포.
외부 API·DB 없이 클라이언트 상태(`useState`)만 사용한 정적 사이트. 동적 OG 이미지(`next/og`),
sitemap·robots 기반 SEO 포함.

## 내가 내린 주요 의사결정

- 민감한 도메인이라 판단해, **"기능을 더하는 것"보다 "위험한 기능을 빼는 것"** 을 우선 —
  AI 상담·위험도 점수화를 의식적으로 제외.
- **데이터 주도 설계** — 상태별 흐름을 데이터로 분기해, 새 상태·기관 추가가 한 곳(`lib/data.ts`,
  `lib/resources.ts`) 수정으로 끝나도록 구성.
- 상담 번호·링크의 **현행성을 직접 웹 검증**하고, 배포 전 재확인 필요성을 README에 명시.
- 사용자 입력(마음 상태·생성 문장)을 **저장·전송하지 않는 원칙**을 코드·문서 전반에 일관 적용.

## 현재 상태 (정직한 기록)

- 설계 → 개발 → 배포까지 완료한 **개인 프로젝트(MVP)**.
- 배포 직후 단계로, **사용자 유입·측정은 진행 전**이다. (방문자 분석 미설치, 홍보 미진행)
- 이 프로젝트의 핵심 가치는 트래픽 수치가 아니라 **민감한 도메인에서의 윤리적 설계 판단**과
  사용자 문턱을 낮추는 UX 설계에 있다.

> ⚠️ 실제 서비스로 운영하려면 각 상담 번호·URL의 현행 운영 여부를 보건복지부·한국생명존중희망재단
> 등 공식 출처로 최종 확인할 것.
